## Data Flow

### 1. Short Audio (< 5 minutes)
```
User Upload → FFmpeg Convert → Whisper Large → Result → Store JSON
     │              │              │              │           │
   10MB         16kHz WAV      5-10 sec       Text      Metadata
```

### 2. Long Audio (5 min - 1 hour)
```
User Upload → FFmpeg Convert → Split Chunks → Process Each → Merge → Store
     │              │              │              │           │        │
   100MB         16kHz WAV    5-min pieces    Whisper      Join    JSON
```

### 3. With Translation
```
Audio → Transcription (Telugu) → Translation API → English/Hindi Text
```

## Security

- File type validation (audio only)
- File size limits (500MB)
- Duration limits (60 min)
- Auto-deletion after 24 hours
- CORS protection
- No persistent user data storage

## Scalability

| Users | RAM Needed | CPU | Strategy |
|-------|-----------|-----|----------|
| 1-5   | 4GB       | 2   | Single instance |
| 5-20  | 8GB       | 4   | Queue + Workers |
| 20+   | 16GB+     | 8+  | Load balancer + GPU |
